## Search Strategy Tips

### Best Practice Sequence:

1. **Go to Sketchfab.com**
2. **Search box**: Enter model name
3. **Filters (right side)**:
   - Format/Type: Leave as "3D Models"
   - License: Look for CC (Creative Commons)
   - Features: Check "Downloadable" box
   - Sort by: "Likes" (most popular first)
4. **Browse results**: Page 1-3 usually has best options
5. **Click model**: Verify:
   - [ ] Has download button
   - [ ] License is visible
   - [ ] Preview looks reasonable
   - [ ] Format is OBJ/GLTF
6. **Download**: Click download → select format (GLTF preferred)

### CGTrader Specific:

1. Go to: https://cgtrader.com/free-3d-models
2. **Search box**: Enter query
3. **Filter "Status"**: Select "Free"
4. **Browse results**
5. **Verify**: Marked "FREE" clearly visible
6. **Download**: Check what formats available

### Quaternius:

1. Go to: https://quaternius.com
2. **Browse sections**: Characters, Nature, Space, etc.
3. **Download free assets**: Many offer free previews/downloads
4. **Note**: May require free account for some

## Success Metrics

✅ **You're on the right track if**:
- Model downloads without errors
- File size < 50 MB
- License is clearly CC0 or CC-BY
- Preview shows recognizable geometry
- Format is GLTF/OBJ

❌ **Red flags**:
- File > 100 MB (likely over-detailed)
- License unclear or commercial restrictions
- Preview shows abstract art rather than 3D model
- Format is unfamiliar (.blend, .max, etc.)
- Requires login/account before download
