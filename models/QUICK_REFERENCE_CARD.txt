## QUICK BLENDER COMMANDS

### Open file and export to GLTF
```bash
blender model.fbx -b -P - << 'EOF'
import bpy
# Scale if needed
bpy.context.active_object.scale = (0.1, 0.1, 0.1)  # Example
# Export
bpy.ops.export_scene.gltf(filepath='model.glb', use_format='GLB')
EOF
```

### Decimate (reduce polycount)
```bash
blender model.blend -b -P - << 'EOF'
import bpy
for obj in bpy.context.scene.objects:
    if obj.type == 'MESH':
        bpy.context.view_layer.objects.active = obj
        mod = obj.modifiers.new(name="Decimate", type='DECIMATE')
        mod.ratio = 0.5  # Keep 50% of vertices
        bpy.ops.object.modifier_apply(modifier="Decimate")
bpy.ops.wm.save_mainfile()
EOF
```
